Antes de empezar

Esta práctica te llevará de la mano desde el archivo vacío hasta una página web funcional. Cada ejercicio construye sobre el anterior.

1

Crea la carpeta del proyecto

Llámala practica-01 en tu escritorio o carpeta de proyectos.

2

Crea dos archivos vacíos

index.html y estilos.css dentro de esa carpeta.

3

Abre VSCode en la carpeta

Arrastra la carpeta al editor o usa File → Open Folder.

4

Instala "Live Server"

En VSCode, extensiones → busca "Live Server" → instalar. Verás cambios en tiempo real.

tip

Con Live Server activo, haz clic derecho en index.html → "Open with Live Server". Tu navegador se actualizará automáticamente cada vez que guardes.

01

La estructura HTML mínima

Abre index.html y escribe (o pega) el siguiente código. No copies y pegues sin leer — entiende cada línea.

index.html
1<!DOCTYPE html> 2<html lang="es"> 3 4 <head> 5 <meta charset="UTF-8"/> 6 <meta name="viewport" content="width=device-width, initial-scale=1.0"/> 7 <title>Mi Práctica</title> 8 <link rel="stylesheet" href="estilos.css"/> 9 </head> 10 11 <body> 12 <!-- Aquí irá tu contenido --> 13 </body> 14 15</html>

¿Qué está pasando en cada línea?

Línea¿Qué hace?
1Declara que esto es un documento HTML5. Va siempre primero.
2Abre el documento. lang="es" dice que está en español.
4–9El <head> contiene configuración invisible. El usuario no la ve.
5Permite usar tildes, ñ y caracteres especiales sin problemas.
6Hace la página adaptable a pantallas de móvil.
7El texto de la pestaña del navegador.
8Vincula el archivo CSS externo. Sin esto, no habrá estilos.
11–13El <body> es donde va TODO lo que el usuario ve.
! ojo

Guarda el archivo (Ctrl+S) y verás una página en blanco en el navegador. Eso es correcto: el <body> está vacío por ahora.

03

Div, id y class

El <div> es un contenedor neutro. Por sí solo no hace nada visual, pero es el bloque de construcción más usado para organizar y estilizar con CSS.

index.html — contenedores
1<!-- id: identificador ÚNICO (solo uno por página) --> 2<div id="cabecera"> 3 <h1>Título del sitio</h1> 4</div> 5 6<!-- class: puede repetirse en muchos elementos --> 7<div class="tarjeta"> 8 <h3>Proyecto A</h3> 9 <p>Descripción del proyecto.</p> 10</div> 11 12<div class="tarjeta"> 13 <h3>Proyecto B</h3> 14 <p>Descripción del proyecto.</p> 15</div>
AtributoRegla de usoEn CSS se llama con
id="nombre"Único por página. Para anclas y JS.#nombre
class="nombre"Reutilizable. Para estilos repetidos..nombre
04

Tu primer CSS

Abre estilos.css. Aquí defines el aspecto visual usando selectores y propiedades.

05

El Box Model

Todo elemento HTML es una caja. Entender el box model es el concepto más importante de CSS.

estilos.css — box model
1.caja-ejemplo { 2 3 /* CONTENIDO: tamaño del área de texto/imagen */ 4 width: 300px; 5 height: auto; 6 7 /* PADDING: espacio INTERNO (entre contenido y borde) */ 8 padding: 1.5rem; /* todos los lados igual */ 9 /* padding: 1rem 2rem; */ /* arriba/abajo izq/der */ 10 11 /* BORDER: la línea del borde */ 12 border: 2px solid #c84b31; 13 14 /* MARGIN: espacio EXTERNO (separa de otros elementos) */ 15 margin: 1rem 0; /* 1rem arriba/abajo, 0 lados */ 16 17}
truco

Abre DevTools en Chrome (F12), haz clic en cualquier elemento y ve a la pestaña Styles → Box Model. Verás visualmente el padding, border y margin de ese elemento.

06

Colores, tipografía y fondos

Practica estas propiedades en tu estilos.css. Cambia los valores y observa el resultado:

estilos.css — colores y tipografía
1body { 2 background-color: #f5f2ec; /* hexadecimal */ 3 /* background-color: rgb(245,242,236); */ /* RGB */ 4 /* background-color: hsl(40,30%,94%); */ /* HSL */ 5 6 font-family: 'Georgia', serif; 7 font-size: 16px; 8 line-height: 1.6; 9} 10 11#cabecera { 12 /* Fondo con gradiente */ 13 background: linear-gradient(135deg, #1a1a1a 0%, #c84b31 100%); 14 color: white; 15 padding: 3rem 2rem; 16 text-align: center; 17} 18 19a { 20 color: #c84b31; 21 text-decoration: none; 22} 23 24a:hover { 25 text-decoration: underline; 26 color: #a03020; 27} 28 29/* Variables CSS: define una vez, usa en todo el archivo */ 30:root { 31 --color-principal: #c84b31; 32 --color-fondo: #f5f2ec; 33 --fuente-base: 'Georgia', serif; 34} 35 36/* Uso de la variable: */ 37h1 { color: var(--color-principal); }
ejercicio

Prueba cambiar #c84b31 por #2c5f8a (azul oscuro) o #2d7a4f (verde). Verás cómo una sola variable puede rediseñar toda la página.
