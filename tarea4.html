// Fundamentos

¿Qué es CSS3?

CSS (Cascading Style Sheets) es el lenguaje que controla la apariencia visual de las páginas web. Mientras HTML define la estructura del contenido, CSS decide cómo se ve: colores, fuentes, tamaños, espaciados, animaciones y mucho más.

CSS3 es la versión actual, que introdujo características revolucionarias como animaciones nativas, flexbox, CSS Grid, variables, gradientes, sombras y media queries para diseño responsivo.

// Selectores

Selectores CSS

Los selectores son el primer paso en cualquier regla CSS: indican a qué elemento(s) HTML se aplicarán los estilos. Dominar los selectores es dominar CSS.

pSelector de etiqueta
.claseSelector de clase
#idSelector de ID
*Universal (todos)
div pDescendiente
div > pHijo directo
h1 + pHermano adyacente
h1 ~ pHermanos generales
a:hoverPseudoclase
p::beforePseudoelemento
[type="text"]Atributo
:not(.clase)Negación
:nth-child(2)Enésimo hijo
:first-childPrimer hijo
:focusCon foco activo
:checkedInput marcado
 selectores.css
/* 1. Selectores básicos */
h1         { color: purple; }        /* etiqueta */
.tarjeta   { padding: 20px; }     /* clase */
#logo      { width: 100px; }      /* id */
*          { box-sizing: border-box; }

/* 2. Combinadores */
nav a      { text-decoration: none; }  /* descendiente */
ul > li    { list-style: none; }      /* hijo directo */

/* 3. Pseudoclases */
a:hover        { color: cyan; }
li:nth-child(2n) { background: #f0f; } /* pares */
input:focus    { border-color: green; }

/* 4. Pseudoelementos */
p::first-line  { font-weight: bold; }
.btn::after    {
  content: " →";
  color: currentColor;
}

Especificidad: cuando dos reglas apuntan al mismo elemento, gana la de mayor especificidad:

1,0,0 ID (#id)
0,1,0 Clase (.clase)
0,0,1 Etiqueta (p, h1)

// Box Model

Modelo de Caja

Todo elemento HTML es una caja rectangular. El Modelo de Caja (Box Model) define cómo se calcula el tamaño y el espacio de cada elemento. Es el concepto más fundamental de CSS.

 box-model.css
/* El modelo de caja completo */
.caja {
  /* CONTENIDO */
  width:   300px;
  height:  200px;

  /* RELLENO interno (padding) */
  padding:        20px;           /* todos los lados */
  padding-top:    10px;
  padding-right:  20px;
  padding-bottom: 10px;
  padding-left:   20px;

  /* BORDE */
  border:         2px solid purple;
  border-radius:  12px;

  /* MARGEN externo (margin) */
  margin:         16px;           /* todos */
  margin:         16px 24px;     /* arriba/abajo  izq/der */
  margin:         0 auto;         /* centrar horizontalmente */

  /* IMPORTANTE: incluye padding y borde en width/height */
  box-sizing: border-box;
}

// Propiedades de texto

Tipografía & Color

Las propiedades de tipografía y color son las más visibles de CSS. Con ellas controlas fuentes, tamaños, pesos, colores y todos los aspectos del texto.

colorTexto

Color del texto. Acepta nombres, HEX, RGB, HSL y RGBA (con opacidad).

color: #8b5cf6;
color: rgb(139,92,246);
color: hsl(262,83%,58%);
font-familyFuente

Define la fuente del texto. Siempre incluye una fuente de respaldo genérica al final.

font-family: 'Nunito', sans-serif;
font-family: 'Courier', monospace;
font-sizeTamaño

Tamaño del texto. px es absoluto, em y rem son relativos al padre o a la raíz.

font-size: 16px;
font-size: 1rem; /* relativo */
font-size: 1.5em; /* al padre */
font-weightGrosor

Grosor de la fuente: de 100 (muy fino) a 900 (muy grueso). bold = 700, normal = 400.

font-weight: 400; /* normal */
font-weight: 700; /* negrita */
font-weight: bold;
line-heightInterlineado

Espacio entre líneas de texto. Un valor sin unidad (como 1.6) es relativo al font-size.

line-height: 1.6;
line-height: 24px;
line-height: 150%;
text-alignAlineación

Alineación horizontal del texto dentro de su contenedor.

text-align: left;
text-align: center;
text-align: right; justify;
text-decorationDecoración

Agrega o quita subrayado, tachado o línea encima del texto.

text-decoration: none;
text-decoration: underline;
text-decoration: line-through;
letter-spacingEspaciado

Espacio entre letras. Valores positivos las separan, negativos las acercan.

letter-spacing: 0.1em;
letter-spacing: 2px;
letter-spacing: -1px;

// Fondos y bordes

Fondos, Bordes & Sombras

CSS3 amplió enormemente las opciones de fondos con gradientes, imágenes múltiples y control total del posicionamiento, además de bordes redondeados y sombras nativas.

 fondos-bordes.css
.elemento {
  /* FONDOS */
  background-color:    #8b5cf6;
  background-image:   url("imagen.jpg");
  background-size:    cover;       /* cover | contain | 100% */
  background-position: center;
  background-repeat:  no-repeat;

  /* GRADIENTES (CSS3) */
  background: linear-gradient(
    135deg,
    #8b5cf6 0%,
    #06b6d4 100%
  );
  background: radial-gradient(
    circle at center,
    #10b981, #06b6d4
  );

  /* BORDES */
  border:        2px solid #8b5cf6;
  border-radius: 12px;           /* redondeado */
  border-radius: 50%;            /* círculo */

  /* SOMBRAS */
  box-shadow:    0 4px 20px rgba(139,92,246,.35);
  box-shadow:    4px 4px 0 #8b5cf6;  /* hard shadow */
  text-shadow:   2px 2px 8px rgba(0,0,0,.3);

  /* OPACIDAD Y TRANSPARENCIA */
  opacity:       0.8;
  background:    rgba(139,92,246, 0.2);
}

// Layout

Flexbox — Diseño en 1D

Flexbox (Flexible Box Layout) es el sistema de diseño unidimensional de CSS3. Permite alinear y distribuir elementos en una fila o columna de forma increíblemente sencilla. Es la herramienta más usada para centrar y distribuir elementos hoy en día.

 flexbox.css
/* CONTENEDOR FLEX */
.contenedor {
  display:         flex;

  /* Dirección */
  flex-direction:  row;           /* row | column | row-reverse */

  /* Alineación principal (eje horizontal si row) */
  justify-content: center;       /* flex-start | flex-end | space-between | space-around */

  /* Alineación cruzada (eje vertical si row) */
  align-items:     center;       /* flex-start | flex-end | stretch | baseline */

  /* Espaciado entre items */
  gap:             16px;

  /* Wrap: permitir que los items pasen a nueva línea */
  flex-wrap:       wrap;
}

/* ITEMS FLEX */
.item {
  flex:      1;             /* crecer y encoger equitativamente */
  flex-grow: 2;             /* este item ocupa el doble */
  order:     -1;            /* reordenar sin cambiar HTML */
  align-self: flex-end;    /* alineación individual */
}

/* CENTRAR PERFECTAMENTE (el truco más famoso de CSS) */
.centrado {
  display:         flex;
  justify-content: center;
  align-items:     center;
  min-height:      100vh;
}

// Layout 2D

CSS Grid — Diseño en 2D

CSS Grid es el sistema de diseño bidimensional más poderoso de la web. Permite crear layouts complejos de filas Y columnas simultáneamente, algo imposible con Flexbox. Es ideal para la estructura general de la página.

 css-grid.css
/* CONTENEDOR GRID */
.grid {
  display: grid;

  /* Definir columnas */
  grid-template-columns: 1fr 1fr 1fr;  /* 3 columnas iguales */
  grid-template-columns: repeat(3, 1fr);   /* igual que arriba */
  grid-template-columns: 200px 1fr 100px;

  /* Columnas responsivas automáticas */
  grid-template-columns: repeat(
    auto-fill,
    minmax(280px, 1fr)
  );

  /* Espaciado */
  gap:      20px;
  row-gap:  16px;
  column-gap: 24px;
}

/* ITEMS: ocupar varias celdas */
.item-grande {
  grid-column: 1 / 3;   /* de columna 1 a 3 (ocupa 2) */
  grid-row:    1 / 3;   /* de fila 1 a 3 */
  grid-column: 1 / -1;  /* de inicio hasta el final */
}

/* LAYOUT COMPLETO DE PÁGINA */
.pagina {
  display: grid;
  grid-template-areas:
    "header  header"
    "sidebar main  "
    "footer  footer";
  grid-template-columns: 250px 1fr;
}
header { grid-area: header; }
aside  { grid-area: sidebar; }
main   { grid-area: main; }
footer { grid-area: footer; }

// Animaciones

Transiciones & Animaciones

CSS3 introdujo animaciones y transiciones nativas, eliminando la necesidad de JavaScript para efectos visuales sencillos. Las transitions animan cambios de estado, mientras que los @keyframes permiten animaciones complejas y continuas.

 animaciones.css
/* ── TRANSICIONES ── */
.boton {
  background:  #8b5cf6;
  transform:   translateY(0);

  /* propiedad | duración | easing | retardo */
  transition:  all .3s ease 0s;
  /* o por separado: */
  transition:  background .2s, transform .3s cubic-bezier(.34,1.56,.64,1);
}
.boton:hover {
  background:  #7c3aed;
  transform:   translateY(-4px);
  box-shadow:  0 8px 24px rgba(139,92,246,.4);
}

/* ── TRANSFORM (sin animar) ── */
.transformado {
  transform: translate(20px, 10px)
              rotate(45deg)
              scale(1.2)
              skew(10deg);
}

/* ── @KEYFRAMES ── */
@keyframes flotar {
  0%   { transform: translateY(0px); }
  50%  { transform: translateY(-15px); }
  100% { transform: translateY(0px); }
}

@keyframes aparecer {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}

.flotante {
  /* nombre | duración | easing | retardo | iteraciones | dirección */
  animation: flotar 3s ease-in-out 0s infinite alternate;
}
.entrada {
  animation: aparecer .6s ease both;
}

// Variables y responsivo

Variables CSS & Media Queries

Las variables CSS (custom properties) permiten definir valores reutilizables, como una paleta de colores central. Las media queries hacen que el diseño se adapte a cualquier tamaño de pantalla.

 variables-responsive.css
/* ── VARIABLES CSS (Custom Properties) ── */
:root {
  /* Paleta de colores */
  --primario:   #8b5cf6;
  --secundario: #06b6d4;
  --acento:     #10b981;
  --texto:      #334155;

  /* Espaciados */
  --espaciado-sm: 8px;
  --espaciado-md: 16px;
  --espaciado-lg: 32px;

  /* Fuentes */
  --fuente-base:  'Nunito', sans-serif;
  --radio:        12px;
}

/* USO de variables */
.boton {
  background:    var(--primario);
  padding:       var(--espaciado-sm) var(--espaciado-md);
  border-radius: var(--radio);
  font-family:   var(--fuente-base);
}

/* ── MEDIA QUERIES ── */
/* Móvil primero (mobile-first) */
.grid { grid-template-columns: 1fr; }

@media (min-width: 640px) {
  /* Tablet */
  .grid { grid-template-columns: repeat(2, 1fr); }
}

@media (min-width: 1024px) {
  /* Escritorio */
  .grid { grid-template-columns: repeat(3, 1fr); }
}

@media (prefers-color-scheme: dark) {
  /* Modo oscuro automático */
  :root { --texto: #f1f5f9; --bg: #0f172a; }
}

@media (max-width: 480px) {
  /* Solo móvil */
  .hero-title { font-size: 2rem; }
  .sidebar   { display: none; }
}

// Buenas prácticas

Código CSS Profesional

Escribir buen CSS no solo es hacerlo funcionar — es hacerlo mantenible, escalable y entendible por otros (¡y por tu yo del futuro!).

  • Mobile-first siempre. Diseña primero para pantallas pequeñas y expande con media queries. Es más sencillo agregar que quitar.
  • Usa variables CSS (custom properties). Define colores, fuentes y espaciados en :root y reutiliza con var(). Un cambio en un lugar lo cambia todo.
  • box-sizing: border-box en todo. Aplica * { box-sizing: border-box; } al inicio de todo proyecto para que los tamaños sean predecibles.
  • Evita !important. Si lo usas seguido, hay un problema de especificidad en tu arquitectura CSS. Reorganiza tus selectores en su lugar.
  • Organiza el CSS por secciones. Usa comentarios tipo /* ── HEADER ── */ para separar el código y facilitar la navegación.
  • Prefiere flexbox/grid sobre posicionamiento absoluto. Los layouts modernos son más limpios y responsivos con display:flex o display:grid.
  • Usa unidades relativas. Prefiere rem para tipografía y % o fr para anchos. Las unidades px son rígidas en pantallas de diferente densidad.
  • No repitas código. Si aplicas el mismo bloque a 5 elementos, crea una clase compartida o usa una variable.
  • Valida con Can I Use. Verifica el soporte de nuevas propiedades CSS en caniuse.com antes de usarlas en producción.
  • Usa las DevTools del navegador. Inspecciona, prueba y modifica CSS en tiempo real con F12. Es la mejor forma de aprender y depurar.

CSS no es difícil — es un lenguaje de reglas con jerarquías claras. Una vez que entiendes la cascada, la especificidad y el modelo de caja, todo lo demás tiene sentido.
